# Windows Portfolio Website 🖥️

A personal portfolio website designed to look and work like a 
Windows desktop environment. Built with pure HTML, CSS and Javascript.

## Live Demo

🌐 [View live site](http://YOUR-S3-BUCKET-URL-HERE) Will Place it Later

## Features

- Draggable windows (About Me, Projects, Resume, Contact)
- Working taskbar with app buttons
- Start menu with pinned apps
- Right-click context menu on desktop
- Live clock in taskbar
- Toast notifications

Just open `MyWeb.html` `MyWeb.css` in any browser. No setup needed.
